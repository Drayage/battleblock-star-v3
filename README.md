# Battle Block Star v3.0

블럭 퍼즐게임 기반 1v1 대전 액션에 로그라이크 진행을 얹은 정적 웹 프로토타입입니다.

## 실행

브라우저 ES 모듈을 사용하므로 로컬 서버로 실행하세요.

```powershell
python -m http.server 5173
```

그 다음 <http://localhost:5173> 을 엽니다.

## 구현된 v3.0 기능

- Canvas 렌더링 엔진과 게임 로직을 `src/` 모듈로 분리
- 기본 `10 x 20` 보드와 HP 행 확장 시스템
- 전투 종료 후 플레이어 필드가 다음 전투로 이어지는 지속성
- 기본 7-Bag 3세트, 총 21장 기반 덱 순환
- 특수 블록 덱 추가: 고화력 I, 기형 십자, 폭탄 O, 마나 T, 정화 O
- 칸 단위 공격력 계산: 기본 0.1, 고화력 0.3
- 총 20라운드 적 선택형 로그라이크 루프
- 5R, 10R, 15R 체크포인트 상점
- 골드, 보상, 스킬 획득 및 최대 3개 장착 슬롯
- PC 키보드 및 모바일 터치 UI

## 주요 조작

- 이동: `←`, `→`
- 소프트 드롭: `↓`
- 회전: `↑`, `X`, `Z`
- 하드 드롭: `Space`
- 홀드: `C`
- 스킬: `1`, `2`, `3`

## 구조

- `index.html`: 앱 진입점
- `src/game.js`: 런/전투/화면 흐름
- `src/board.js`: 보드, HP 행, 라인 클리어, 공격력 계산
- `src/deck.js`: 21장 7-Bag 덱
- `src/progression.js`: 라운드, 적, 보상, 상점
- `src/skills.js`: 장착형 스킬
- `src/renderer.js`: Canvas 렌더러
- `src/input.js`: 키보드 및 터치 입력
